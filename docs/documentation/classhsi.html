Detailed Description

hsi is a class that holds the hyperspectral image.

It is used to contain both the hyperspectral image and additional information. For labels, use hsiInfo class.

◆ GetFgMask()

function GetFgMask ( in  obj,
in  varargin 
)

GetFgMask returns the foreground mask for a sample.

The foreground mask corresponds to the tensors that belong to the tissue. The mask is prepared using function RemoveBackgroundInternal. See also https://www.mathworks.com/help/images/color-based-segmentation-using-k-means-clustering.html.

Usage

hsIm = hsIm.GetFgMask();
Parameters
obj[hsi] | An instance of the hsi class
colorLevelsForKMeans[int] | Optional: Color levels for Kmeans. Default is 6.
attemptsForKMeans[int] | Optional: Attempts for Kmeans. Default is 3.
layerSelectionThreshold[double] | Optional: Threshold for layer selection. Default is 0.1.
bigHoleCoefficient[double] | Optional: Coefficient for closing big holes. Default is 1000.
closingCoefficient[double] | Optional: Coefficient for closing operation. Default is 2.
openingCoefficient[double] | Optional: Coefficient for opening operation. Default is 5.
Return values
fgMask[numeric array] | A foreground mask
