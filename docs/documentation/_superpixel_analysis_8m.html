CleanLabels returns superpixel labels that contain tissue pixels.

Keep only pixels that belong to the tissue (Superpixel might assign background pixels also). The last label is background label.

Usage

[cleanLabels, validLabels] = CleanLabels(labels, fgMask, pixelNum);
