ChromophoreAbsorption plots the skin chromophore absorption functions.

Information is saved in config::[importDir]\*.csv. Required .csv files:

  • 'pheomelanin_absroption.csv'
  • 'eumelanin_absroption.csv'
  • 'hb_absorption_spectra_prahl.csv' Values by Scott Prahl (https://omlc.org/spectra/)

Usage [extCoeffEumelanin2, extCoeffHbO, extCoeffHbR] = plots.ChromophoreAbsorption();
